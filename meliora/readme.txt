PLEASE READ

SERVER SIDE:
npm install in the root folder to install server libraries and packages
.../meliora> npm install

nodemon need to be installed globally,
.../meliora> npm install -g nodemon

run the server:
.../meliora> npm start


CLIENT SIDE:
navigate to the client folder to install client libraries and packages
.../meliora> cd client
client> npm install

run the client:
client> npm start
